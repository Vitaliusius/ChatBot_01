# Телеграм-бот уведомляющий о статусе проверки работ.

Данный бот проверяет статус отправленной на ревью работы. Как только статус проверки меняется, бот сразу отправляет статус пользователю в Телеграм.

## Как запустить

Для запуска сайта вам понадобится Python третьей версии.

Скачайте репозиторий. Затем установите зависимости:

```sh
pip install -r requirements.txt
```

Запустите бота командой:

```sh
python main.py
```

## Переменные окружения


Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл `.env` в корневом каталоге. И запишите туда данные как в примере ниже:

Доступны переменные:

- `DEVMAN_TOKEN` = Токен телеграм бота.
- `TELEGRAM_BOT_API_KEY` = API токен  "Devman".
- `TELEGRAM_CHAT_ID` = ID  чата с ботом.
